# SSOT Mission Control Prototype

Root-level SSOT dashboard prototype for Evolution Mission Control.

## Run

```bash
cd /home/evo/projects/SSOT_Build
npm install
npm run dev
```

Open `http://localhost:3000`.

## Live wiring currently enabled

- Loads SSOT snapshot directly from `intake/v0.1/seed.json`
- Horse names link to live breeding pages
- Performance links open live performance profile pages
- Intake queue source URLs are clickable
