# Evolution SSOT Mission Control

Mission-control style dashboard prototype for SSOT intake tracking.

## Run locally

1. Install dependencies:
   `npm install`
2. Start dev server:
   `npm run dev`
3. Open:
   `http://localhost:3000`

## Current data source

The dashboard currently renders from `data/ssotSnapshot.ts`, which mirrors the
`intake/v0.1` snapshot.
