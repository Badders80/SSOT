# Ad Hoc Intake

This folder is the local working area for imported source material before it becomes canonical.

Use it for:

- old source documents
- partial extracts
- OCR output
- notes from manual review
- raw supporting assets that are not yet approved

Rules:

- keep raw source material in the matching template family folder
- do not treat anything here as canonical build truth
- promote reusable content into `data/templates/`, `data/library/`, or `data/assets/` only after review

- `documents/product_disclosure_statement/`
